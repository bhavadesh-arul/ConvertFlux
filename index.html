About ConvertFlux

ConvertFlux is a privacy-first file conversion platform that runs entirely in your browser. We believe your files are yours — and they should never have to leave your device just to be converted.

Our Mission

Most online file converters upload your documents to remote servers. That means your contracts, personal photos, financial spreadsheets, and private data pass through systems you don't control. ConvertFlux flips this model entirely.

Every conversion — whether compressing a PDF, converting an image, or merging spreadsheets — happens locally using your device's computing power. We never see your files. We can't, by design.

How It Works

We use modern browser APIs and WebAssembly-powered libraries (pdf.js, pdf-lib, jsPDF, and others) to perform complex file operations without any server involvement. When you load ConvertFlux, all the processing code is downloaded once and runs locally.

Open & Honest

We don't claim features we don't have. PDF → Word "conversion" with formatting preservation isn't technically possible in the browser, so we offer PDF text extraction instead — which is honest and genuinely useful.

Privacy Policy

Your privacy is not a feature — it's the foundation of ConvertFlux.

Data Collection

We collect zero personal data. ConvertFlux does not use cookies for tracking, does not collect analytics on file types processed, and does not log any user activity beyond what your own browser stores locally.

File Processing

All file processing occurs in your browser using client-side JavaScript. Your files are loaded into memory, transformed, and made available for download — all without leaving your device. Files are never transmitted to any server.

Local Storage

ConvertFlux uses your browser's localStorage to save a history of your last 5 conversions (file names and types only — not file contents). You can clear this at any time using the "Clear All" button in the Recent Conversions section.

Third-Party Services

We load open-source libraries from trusted CDNs (Cloudflare). These CDNs may log the request to load the script, but receive no information about your files or usage.

Contact Form

If you use the contact form, your message is sent via Formspree. Please review Formspree's privacy policy for how they handle submitted form data.
